# Design Pattern Using PHP

- Creational Design Patterns
   - Factory Method
   - Abstract Factory
   - Builder
   - Prototype
   - Singleton

- Structural Design Patterns
   - Adapter [ Wrapper ]
   - Bridge
   - Composite [ Object Tree ]
   - Decorator
   - Facade
   - Flyweight [Cache]
   - Proxy

- Behavioral Design Patterns
   - Chain of Responsibility
   - Command
   - Iterator
